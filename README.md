## About Me

**RTL Design & Verification Engineer** building industry-oriented digital blocks — from RTL and FPGA integration through functional verification, wave-based debug, and verification automation.

I develop synchronous and asynchronous FIFOs, AXI-Stream interfaces, processor datapaths, and CDC-oriented designs, with hands-on flows in **SystemVerilog/Verilog**, **ModelSim**, **GTKWave**, and **Vivado**. My GitHub is my portfolio: block-level RTL, testbench bring-up, and Python-based DV scripting aligned with how silicon teams work.

- 🔧 **RTL & digital architecture** — FSM-centric design, bus/FIFO blocks, and structured RTL for simulation and FPGA flows  
- 🧪 **Functional verification** — SystemVerilog testbenches, stimulus/checkers, linting, and waveform-driven debug  
- ⚙️ **Verification automation** — Python scripting to streamline regression-style checks and DV workflows  
- 📐 **FPGA-oriented integration** — RTL-to-bitstream mindset with Vivado and Linux-based tool environments  
- 🛠️ **Engineering discipline** — Git-tracked projects, reproducible sim setups, and industry-style repo organization
